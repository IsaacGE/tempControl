# tempControl
Sistema de control de temperatura con Arduino
Sistema con modos automático manual
Calefacción simulada por led y aire acondicionado simulado por motor DC

Componentes usados:
  - motor DC
  - LED
  - Sensor analógico de temperatura LM35
  - LCD
  - Potenciometro
 
Librerias usadas:
  - Simulino
  - LCD_12C
